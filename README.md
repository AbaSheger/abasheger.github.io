# Portfolio Website

## Description
My personal portfolio website built with React, showcasing my projects, skills, and professional experience. Visit the live site at [https://abasheger.github.io](https://abasheger.github.io)

## Technologies Used
- React
- Tailwind CSS
- GitHub Pages
- JavaScript/JSX

## Features
- Responsive design
- Dark/Light mode toggle
- Interactive project cards
- Smooth scrolling navigation
- Multi-language support
- Dynamic content loading

## Sections
- About Me
- Projects Portfolio
- Technical Skills
- Contact Information
- CV/Resume
- Education

## Local Development
1. Clone the repository:
```bash
git clone https://github.com/abasheger/abasheger.github.io.git
```

2. Install dependencies:
```bash
npm install
```

3. Start development server:
```bash
npm start
```

4. To deploy:
```bash
npm run deploy
```

## Project Structure
- `/src` - React components and source code
- `/public` - Static assets
- `/src/data` - Content data (projects, translations, skills)
- `/src/components` - React components
